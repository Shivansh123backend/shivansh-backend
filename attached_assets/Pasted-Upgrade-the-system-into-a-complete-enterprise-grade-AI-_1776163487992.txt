Upgrade the system into a complete enterprise-grade AI-powered contact center dialer (Vicidial + VAPI hybrid) with predictive dialing, compliance, agent control, and real-time monitoring.

---

GOAL:

Build a production-ready system that supports:

* Multi-campaign dialing
* Predictive + progressive dialing
* Dialing speed & pacing control
* Agent + number resource management
* Supervisor monitoring (listen, whisper, barge)
* Inbound + outbound routing
* Compliance (DNC + time windows)
* Lead recycling + retry logic
* Spam protection
* Advanced reporting (CDR + analytics)
* Stable queue-based execution

7. PARALLEL + PREDICTIVE DIALING
================================

Calculate:

available_agents = agents available

target_calls = available_agents * dialing_ratio

Limit:

if target_calls > max_parallel_calls:
→ cap

========================
8. DIALING SPEED CONTROL
========================

dialing_speed = calls/sec

Implement batch dialing:

* run every second
* place calls = dialing_speed
* delay 100–300ms

========================
9. DROP RATE CONTROL (CRITICAL)
===============================

Track:

* total_calls
* abandoned_calls

If drop_rate > limit:

→ reduce dialing_speed

========================
10. ADAPTIVE PACING
===================
